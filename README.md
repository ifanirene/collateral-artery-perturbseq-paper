# Collateral Artery Perturb-seq Paper

## Overview

This repository contains code, data, and analysis for the collateral artery formation Perturb-seq study.

## Repository Structure

- `src/` - Source code for analysis and processing
- `notebooks/` - Jupyter notebooks for exploratory analysis
- `data/` - Input datasets (not committed if large)
- `results/` - Analysis outputs and intermediate results
- `figures/` - Generated figures for the manuscript
- `workflow/` - Workflow scripts and pipelines
- `config/` - Configuration files
- `env/` - Environment configuration files
- `Methods.md` - Detailed methods documentation
